Default CMYK ICC for PSD/PSB files that lack an embedded profile (IR 1039).

File: CGATS001Compat-v2-micro.icc
Source: https://github.com/saucecontrol/Compact-ICC-Profiles (MIT)
Why not Japan Color 2001 Coated: Adobe/Japan Color profiles are not freely
redistributable for bundling in applications without a separate bundling
license. This compact CGATS.001-compatible coated CMYK profile is MIT-licensed
and empirically matches Photoshop thumbnail greens on the brochure corpus
far better than naive CMYK->RGB.
