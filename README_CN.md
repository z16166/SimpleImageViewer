# Simple Image Viewer (SimpleImageViewer)

基于 Rust 构建的高性能跨平台图片查看器。专为快速浏览大型图片库而设计，支持多主题界面切换（深色/浅色/跟随系统）、后台音乐播放功能，以及持久化设置。
 
![界面截图](assets/screenshot.jpg)

---

## 功能概要

Simple Image Viewer 是一款轻量、快速的桌面图片查看器。它在后台预加载相邻图片以实现即时翻页，支持自动幻灯片播放，并可在查看图片的同时播放背景音乐。所有用户设置均自动保存到磁盘，下次启动时自动恢复上次的图片目录和音乐。

## 特性列表

- **快速图片加载** — 后台线程预加载前后图片，翻页无延迟。包含针对 Windows 平台大尺寸竖拍（旋转）JPEG 图片的 WIC 管道性能深度优化
- **图像格式支持** — 覆盖常见与现代静态图、Photoshop 文档及多种相机 RAW（RAW 为仅浏览）。
  - **常见静态**：JPEG、PNG、GIF、BMP、TIFF、TGA、WebP、ICO、PNM、QOI
  - **现代与高动态**：JPEG XL（`.jxl`）、AVIF / AVIFS 序列（`.avif`、`.avifs`）、OpenEXR（`.exr`）、Radiance HDR（`.hdr`）、HEIF / HEIC / HIF（含典型 iPhone HEIC）
  - **Photoshop**：PSD / PSB；PSB 大文档加载前有内存安全检查
  - **相机 RAW**（60 余种）：佳能（`.cr2`、`.cr3`）、尼康（`.nef`、`.nrw`）、索尼（`.arw`）、富士（`.raf`）、松下（`.rw2`）、奥林巴斯（`.orf`）、宾得（`.pef`）、哈苏（`.3fr`）、飞思（`.iiq`）等
- **HDR 与高动态渲染** — 对自带 HDR 或更广亮度动态的文件提供 HDR 观感呈现；最终观感取决于 HDR 显示器与系统的 HDR 是否开启。
  - Ultra HDR JPEG、自带 HDR 信息的 JPEG
  - Radiance HDR（`.hdr`）
  - OpenEXR（`.exr`）
  - 保留更高亮度层次的 TIFF
  - JPEG XL
  - AVIF / AVIFS
  - HEIF / HEIC / HIF
- **超大图分片渲染** — 内置分片渲染引擎，支持超高分辨率图片（1亿像素以上）；仅将视口内的切片上传至 GPU，通过高效缓存管理保持显存占用恒定
- **图片打印** — 直接从应用内打印图片。
  - **Windows**: 调用系统原生打印向导。支持高保真 JPEG (95% 质量) 和自动 Alpha 透明度扁平化处理。
  - **macOS / Linux**: 自动将图像导出为完美尺寸、无边距的 PDF，并调用系统默认查看器打开以供打印。
  - **灵活模式**: 支持打印“全图”或仅打印当前缩放后的“可见区域”。
- **主题选择** — 在设置中可实时切换 **深色** (经典)、**浅色** 或 **跟随系统** 主题。
- **系统托盘图标支持** — 选项支持关闭窗口时最小化到系统托盘（跨平台）。支持左键点击托盘图标恢复窗口，右键托盘菜单（显示主窗口、设置、退出），以及在托盘状态下双击图片打开时自动恢复并展示（基于 IPC 机制）。
- **复制/剪切图片到指定目录** — 支持右键菜单或通过自定义快捷键（`Ctrl + Shift + C` 复制，`Ctrl + Shift + X` 剪切/移动）将当前图片保存到指定目录。提供记住上次目录的文件夹选择对话框，剪切成功后自动从列表中移除当前图片（支持跨盘符移动回退机制）。
- **Windows 系统集成** — 在设置面板中可一键注册为 Windows「打开方式」菜单中的推荐图片查看器（无需管理员权限）。提供「关联格式」对话框选择需要关联的文件类型，以及「删除关联」一键清除所有注册表项、还原为绿色软件状态
- **动画图片播放** — 自动播放 GIF 动图、APNG 和 WebP 动画，帧间延迟精确还原
- **流畅导航** — 方向键或 `PageUp`/`PageDown` 翻页、鼠标滚轮缩放、1:1 模式下拖拽平移，`F5` 键一键刷新文件列表且不重置缩放/旋转/播放动画，主窗口按 `Ctrl + O` 可直接选择图片文件夹（无需先打开设置）
- **自定义快捷键** — 在 **设置 > 按键（Hotkeys）** 中重新绑定翻页、缩放、旋转、幻灯片、打印、选择图片目录（`Ctrl + O`）等主要操作；支持键盘快捷键、鼠标滚轮以及带修饰键的鼠标按键。下方 **操作说明** 中的默认键位可随时修改
- **两种缩放模式** — *适应窗口*（默认）和*原始尺寸（1:1）*；按 `Z` 切换
- **EXIF & XMP 元数据查看** — 右键点击图片可查看详细的 EXIF 信息或 XMP 属性；针对常用标签（创作者、版权、工具等）进行了平铺化显示优化
- **像素检查器 (Pixel Inspector)** — 悬停实时查看像素坐标与 RGBA 颜色，或按住 Shift 点击选取自定义区域来查看详细的像素网格值。
- **模态对话框交互** — 元数据窗口、转到、关联设置等对话框升级为“真·模态”；开启时自动暗化背景并拦截底层交互，提供沉浸式操作体验
- **无干扰模式 (OSD)** — 可以在设置中隐藏全部悬浮文字（如图片名、缩放比例等），体验纯净看图
- **记忆播放历史** — 选项支持记忆上次关闭时查看的图片，下次启动后自动从该图开始浏览
- **自动播放幻灯片** — 间隔时间可设置（0.5 秒 – 1 小时），支持循环/到末尾停止和随机顺序播放
- **背景音乐播放** — 高保真音频回放，支持 MP3、FLAC、OGG、WAV、AAC、M4A、**APE**；内置**音频设备抢占自动恢复**机制 (Windows)。**“上一曲”采用车载音响逻辑**：播放时长超过 3 秒时点击为“重播本曲”，3 秒内连续点击（或曲首点击）则跳转到真正的上一首。
- **增强型音乐 OSD (悬浮窗)** — 极简主义悬浮窗设计，自动优先展示来自 CUE 或文件标签的曲目元数据（索引+标题），而非单纯的文件名
- **智能自动消隐** — 音乐悬浮窗在闲置 5 秒后自动隐退，确保图片视界毫无遮挡。一旦鼠标移动或发生音频交互，悬浮窗将即刻唤醒
- **五键交互控制栏** — 设置面板内置紧凑的控制栏（⏮ ⏪ ▶/⏸ ⏩ ⏭），支持在物理音乐文件与逻辑 CUE 曲目之间快速跳转
- **实时音量调节** — 设置面板内提供滑动条，音量在会话间持久保存
- **递归目录扫描** — 可选扫描所有子文件夹中的图片
- **目录树导航** — 在嵌入式侧栏或独立窗口中浏览文件夹，列表带缩略图预览与可排序列；可快速跳转到桌面、文档、盘符与网络位置
- **双击查看图片时不改变图库目录** — 图库设置中可开启：从资源管理器双击打开图片时，本次浏览会使用该图片所在目录，但不会覆盖已保存的图库目录；通过 **选择** 按钮或目录树选择的目录仍会被记住
- **跳过成对 RAW** — 图库设置中可开启：当同目录存在同名 JPG/JPEG 时跳过 RAW 文件（例如索尼相机 RAW+JPEG 同拍模式），浏览时无需反复解码 RAW，更省资源
- **RAW 去马赛克模式（CPU / GPU）** — 在 **设置 > 显示选项** 中开启「高质量 RAW 预览」后，可选择 **CPU** 或 **GPU** 去马赛克。GPU 模式对标准 Bayer RAW 使用计算着色器加速；富士 X-Trans、Super CCD、非方形像素（如尼康 D1X）、超大图或处理失败时自动回退 CPU，OSD 会显示实际执行路径。RAW 默认亮度因机型与场景而异，请根据需要按 `Ctrl + ↑` / `Ctrl + ↓` 手动调节曝光（EV）。
- **设置桌面壁纸**：右键点击任何图片即可将其设置为壁纸，支持多种布局模式（填充、适应、拉伸、平铺、居中、跨显示器）。
- **氛围感转场动画**：提供多种专业的双纹理过渡效果，包括 **Cross-Fade (淡入淡出)**、**Zoom & Fade (缩放淡入)**、**Slide (滑入)**、**Push (推入)**、**Page Flip (翻页)**、**Ripple (水纹扩散)**、**Curtain (拉幕)** 以及 **Random (随机)** 模式。支持 50ms 到 2000ms 的时长调节。
- **持久化设置** — 所有偏好设置保存到可执行文件旁边的 `siv_settings.yaml`，下次启动时自动加载
- **会话恢复** — 记住上次的图片目录和音乐路径，启动时自动加载
- **全屏模式** — 按 `F11` 或 `F` 切换；程序始终以窗口模式启动（显示系统标题栏）
- **现代化 UI 面板** — 采用紧凑的双排分布设置面板，支持一键点击主窗口空白背景快速关闭面板
- **字体选择与大小调节** — 可从系统字体库中选择 UI 字体，并自由调节界面缩放（12–32 像素）
- **图片预加载开关** — 可选择禁用后台相邻图片预加载，以节省系统资源
- **快速跳转** — 按 `G` 打开“跳转到图片...”对话框，可直接输入索引跳转到任意图片。
- **智能格式检测** — 即使图片文件的后缀名名实不符（例如 JPEG 文件被错误命名为 `.png`），程序也会自动识别其实际格式并加载，有效解决因文件扩展名标记错误导致的加载失败问题。
- **文件删除** — 按 `Delete` 键将当前图片移至回收站，或按 `Shift + Delete` 永久删除（为了效率，不设确认对话框）。
- **右键菜单** — 在图片上点击右键可使用复制路径、复制文件、查看 EXIF/XMP、设为桌面墙纸、旋转、打印等内置操作
- **自定义右键菜单** — 在 **设置 > 右键菜单（Context Menu）** 中可调整内置项顺序、插入分隔线、显示或隐藏条目，并添加自定义操作（启动可执行文件或完整命令行，`%1` 表示当前图片路径）
- **多语言支持 (i18n)** — 界面自动适应系统语言（英语、简体中文、繁体中文 - 台湾和香港），支持手动切换及完美的回退机制
- **增强型崩溃防御与系统诊断** — 内置全局异常监控机制。在发生严重错误时提供本地化对话框反馈，并支持自动诊断快照捕获与持久化日志记录，极大提升了故障反馈与排查效率

### 使用流程

1. 启动程序，按 `Ctrl + O` 直接选择图片目录，或按 `F1` 打开设置面板后点击 **📁 选择**
2. 上次使用的目录会自动恢复；扫描完成后设置面板自动关闭（若通过 `F1` 打开）
3. 使用方向键或鼠标滚轮浏览图片
4. 如需背景音乐：勾选"Play background music"，点击 **🎵 File** 或 **📂 Dir** 选择音乐文件/目录

### Linux 下的 HDR

Linux 原生 HDR 需要：**Wayland 会话**（X11 不支持）、**HDR 显示器**且在桌面环境中已开启 HDR（如 KDE Plasma），以及在 **设置（`F1`）> 显示选项** 中启用 **要求原生 HDR 表面**（若提示需重启请重启应用）。

没有原生 HDR 时仍可正常查看 HDR 图片，程序会回退到 **SDR 色调映射**；可用 `Ctrl + ↑` / `Ctrl + ↓` 调节曝光（EV）。

**何时需要 [vk-hdr-layer](https://copr.fedorainfracloud.org/coprs/jackgreiner/vk-hdr-layer/) + `ENABLE_HDR_WSI=1`**

在 Wayland 下，显卡/驱动须向 Vulkan 暴露 HDR 交换链。**NVIDIA** 上在专有驱动尚未内置 HDR WSI 扩展时，通常需要社区 **vk-hdr-layer**（详见下表），并以如下方式启动：

```bash
ENABLE_HDR_WSI=1 ./SimpleImageViewer
```

Fedora 示例：

```bash
sudo dnf copr enable jackgreiner/vk-hdr-layer
sudo dnf install vk-hdr-layer
ENABLE_HDR_WSI=1 ./SimpleImageViewer
```

**NVIDIA RTX 20xx 及更新（Turing 及以后）** 使用专有驱动 **595+**（如 RTX 50 系列）时，一般**无需** vk-hdr-layer 或 `ENABLE_HDR_WSI`。**550–590** 主线驱动上，配合 vk-hdr-layer + `ENABLE_HDR_WSI=1` 也常有原生 HDR。**Maxwell / Pascal / Volta** 显卡（如 GTX 750–1080 Ti）须使用 **580.xx 旧版（legacy）** 驱动分支，且通常需要 vk-hdr-layer — 无法安装 595+ 或 590+ 主线驱动。AMD / Intel 使用 Mesa 时，满足下表版本通常无需上述步骤。

**已验证可用的环境**

| 显卡 | 驱动 / 栈 | 说明 |
|---|---|---|
| AMD（GCN 3 及更新） | Mesa RADV 25.1+ | |
| Intel（第 9 代及更新） | Mesa ANV 25.1+ | |
| NVIDIA Turing 及更新（RTX 20xx 及以后） | 专有驱动 **595+** 可免 vk-hdr-layer；**550–590** 常需 vk-hdr-layer + `ENABLE_HDR_WSI=1` | |
| NVIDIA Maxwell / Pascal / Volta（GTX 750–1080 Ti） | **580.xx legacy** 分支 + vk-hdr-layer + `ENABLE_HDR_WSI=1` | 无法使用 595+ 或 590+ 主线驱动 |

查看 OSD 中的 HDR 行：**原生 Wayland HDR** 表示已走原生呈现；**SDR 色调映射** 表示回退路径（NVIDIA 上可尝试 vk-hdr-layer / `ENABLE_HDR_WSI=1`）。

## 设置文件说明

首次运行后，`siv_settings.yaml` 会自动生成在可执行文件旁边：

```yaml
recursive: false
last_image_dir: "D:\\Photos"
auto_switch: true
auto_switch_interval: 5.0
loop_playback: true
scale_mode: fit_to_window
play_music: true
music_path: "D:\\Music"
volume: 0.8
music_paused: false
show_music_osd: true
last_music_track: "D:\\Music\\Album\\CD1.flac"
font_family: "Microsoft YaHei"
```
